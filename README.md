#   **FooData - Site de Recettes**

FooData est un site de recettes réalisé avec React.js. Il vous permet de rechercher des recettes de plats en utilisant l'API "The Meal DB" et d'ajouter vos propres recettes personnalisées.

###  **Fonctionnalités :**

- Affichage des dernières recettes à partir de l'API "The Meal DB".
- Détails de chaque recette avec instructions, catégorie, et région.
- Possibilité d'ajouter vos propres recettes personnalisées.

###  **Utilisation**

####   Page d'accueil : 
La page d'accueil affiche les dernières recettes à partir de l'API "The Meal DB". Vous pouvez également ajouter vos propres recettes en cliquant sur le lien "Ajouter une recette".

####   Page de détails de la recette :
Pour voir les détails d'une recette, cliquez sur le nom de la recette depuis la page d'accueil. Vous verrez les instructions, la catégorie, la région, et une image de la recette.

####   Page d'ajout de recette personnalisée :
 Cliquez sur "Ajouter une recette" depuis la page d'accueil pour ajouter vos propres recettes. Remplissez les informations demandées et soumettez le formulaire.


###  **API "The Meal DB"**

Ce projet utilise l'API "The Meal DB" pour récupérer des recettes de plats. Vous pouvez en savoir plus sur cette API en visitant leur site web : https://www.themealdb.com/api.php
